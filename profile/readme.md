# LM01 Software Engineer Group-1 Projects

Welcome to the official repository organization for Group 1 (Class LM01). This organization hosts the source code for our Software Engineering course project: a modern, self-service Internet Cafe (Warnet) Management System.

## Project Overview
A full-stack solution designed to streamline internet cafe operations. It features self-service customer accounts, real-time PC status tracking, stackable time-based billing, and an integrated food and beverage menu.

## 📂 Repositories
* **[Backend API](https://github.com/SE-Elysia1/SE-elysia1):** The core server handling user authentication, session timers, and database management.
* **Frontend Client:** *Work In Progress (WIP)* — The user interfaces for the Admin Dashboard and the Customer PC portals.

## 🛠️ Tech Stack

**Backend System:**
* **Runtime:** Bun
* **Framework:** ElysiaJS
* **Database:** SQLite
* **ORM:** Drizzle ORM
